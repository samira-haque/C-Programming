## 🚀 Getting Started

### Prerequisites

- GCC Compiler
- Code::Blocks / VS Code / Dev-C++ / Turbo C (optional)

### Clone the Repository

```bash
git clone https://github.com/samira-haque/C-Programming.git
cd C-Programming
```

### Compile and Run

```bash
gcc filename.c -o output
./output
```

#### Windows

```bash
gcc filename.c -o output.exe
output.exe
```
